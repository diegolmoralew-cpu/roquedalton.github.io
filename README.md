# Roque Dalton — El poeta prohibido

Página web educativa sobre la vida y obra del poeta salvadoreño **Roque Dalton García** (1935–1975), con énfasis en su obra *Las historias prohibidas del Pulgarcito* (1974).

## Contenido

- Biografía de Roque Dalton
- Cronología de su vida
- Análisis de *Las historias prohibidas del Pulgarcito*
- Legado literario
- Código QR de acceso a la página

## Tecnologías

- HTML5 / CSS3 puro (sin frameworks)
- [QRCode.js](https://cdnjs.cloudflare.com/ajax/libs/qrcodejs/1.0.0/qrcode.min.js) para generación del QR
- Google Fonts: Playfair Display, Crimson Text, Courier Prime
